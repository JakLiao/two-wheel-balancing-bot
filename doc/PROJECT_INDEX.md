# 两轮平衡车 PID 学习项目

> 野火小智 STM32F103C8T6 · TB6612FNG · MPU6050 · HC-05

---

## 项目结构

```
two-wheel-balancing-bot/
├── doc/
│   ├── HARDWARE_SPEC.md        ✅ 硬件规格 + 引脚定义
│   └── (more docs)
├── src/
│   ├── app/
│   │   ├── main.c              ← 入口
│   │   ├── balance.c/h         ← 平衡控制核心（直立环 + 速度环）
│   │   └── bluetooth.c/h       ← HC-05 指令解析
│   ├── drivers/
│   │   ├── mpu6050.c/h         ← MPU6050 驱动（I2C，DMP 可选）
│   │   ├── encoder.c/h         ← 编码器测速（TIM2 正交解码）
│   │   ├── motor.c/h           ← TB6612 + PWM 控制
│   │   └── exti.c/h            ← 外部中断配置
│   ├── middleware/
│   │   └── pid.c/h             ← 通用 PID 控制器
│   └── utils/
│       └── filter.c/h           ← 一阶互补滤波 /卡尔曼
├── platformio.ini              ← (待确认开发环境后创建)
└── test/
```
