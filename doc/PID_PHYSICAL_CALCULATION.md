## 二、Ziegler-Nichols 临界比例度法推导

### Step 1：系统固有频率估算（倒立摆简化模型）

```
两轮平衡车倒立摆固有角频率（简化单摆模型）：

等效摆长 h ≈ 35 cm = 0.35 m（重心到轮轴）

ω₀ = √(g/h) = √(9.8 / 0.35) ≈ 5.29 rad/s
T₀ = 2π/ω₀ ≈ 1.19 s

（轮子纯滚动无滑动约束下的特征周期）
```

### Step 2：系统转动惯量估算

```
车身质量 m ≈ 1.5 kg（粗略估计）

绕轮轴的转动惯量（简化圆柱体模型）：
I = (1/3) × m × h² = (1/3) × 1.5 × 0.35² ≈ 0.061 kg·m²

（实际平衡车更接近细长杆绕一端旋转，I = (1/3)ml²）
```

### Step 3：电机力矩常数换算

```
电机规格：起动转矩 2.6 kg·cm = 0.255 N·m（输出轴）
减速比：20.4 : 1

电机轴等效扭矩常数（估算）：
Kt = 0.255 N·m / 2.67A ≈ 0.0955 N·m/A

PWM=100 时的最大输出扭矩（输出轴）：
T_max = 0.255 N·m

等效推进力：
F_max = T_max / r = 0.255 / 0.03 = 8.5 N
车身重量 ≈ 15 N → 最大加速度 ≈ 0.57g
```

### Step 4：临界增益 Ku 估算

```
阻尼比 ζ 估算（倒立摆系统特征）：
ζ ≈ 0.10 ~ 0.20（欠阻尼，直立平衡车自然特征）

取 ζ = 0.15

ω_damped = ω₀ × √(1-ζ²) ≈ 5.29 × 0.988 ≈ 5.23 rad/s

临界增益（线性化倒立摆传递函数推导）：
  G(s) = (g - kh) / (I/m + mgh²)
  其中 k = 电机等效刚度

Ku（归一化到 PWM 量程 100）≈ 6.4

Ziegler-Nichols 临界比例度法参数表（欠阻尼修正）：
┌────────┬──────────┬───────────┬───────────┐
│  类型   │    Kp    │    Ki     │    Kd     │
├────────┼──────────┼───────────┼───────────┤
│   P    │  Ku/2    │     0     │     0     │
│   PI   │  Ku/2.2  │  Ku/8.8   │     0     │
│  PID  │  Ku/1.5  │  Ku/3.3   │  Ku/2.6   │
└────────┴──────────┴───────────┴───────────┘

→ Z-N PID 参考值：Kp≈6.4, Ki≈1.9, Kd≈2.5
```

### Step 5：江协科技参考值换算（关键桥梁）

> 江协科技参考 Kp=200~700 基于 **PWM 量程 7200**（ARR=7199）
> 本车 PWM 量程是 **100**

```
换算系数 = 100 / 7200 ≈ 0.0139

Kp 参考范围（换算到本车量程）：
  最小：200 × 0.0139 ≈ 2.8
  最大：700 × 0.0139 ≈ 9.7

→ 江协科技换算推荐：Kp = 2.8 ~ 9.7
→ 本车当前 Kp=6.0，正好在中间 ✅
```

## 四、Kp/Kd/Ki 过大过小判断标准

> 来源：江协科技 + 物理推导双重验证

### Kp（比例增益）

| 状态 | 现象 | 应对 |
|------|------|------|
| **Kp 过小** | 推一下，车慢慢回正（响应迟钝）或不回正倒地 | ↑ 增大 Kp |
| **Kp 合适** | 推一下 → 车快速回正，1~2 次内停 | ✅ 最优 |
| **Kp 偏大** | 低频来回摆动（0.5~2Hz），2~3 次才停 | ↓ 减小 Kp |
| **Kp 过大** | 持续低频振荡，越来越厉害，直到倒地 | 立刻 ↓ 减小 Kp |

### Kd（微分增益）

| 状态 | 现象 | 应对 |
|------|------|------|
| **Kd 过小** | 推一下后，来回振荡 3 次以上才停 | ↑ 增大 Kd |
| **Kd 合适** | 推一下 → 回正干脆，无多余振荡 | ✅ 最优 |
| **Kd 偏大** | 静止时有细微高频颤动，电机有轻微啸叫 | ↓ 减小 Kd |
| **Kd 过大** | 剧烈高频抽搐（⚠️会损伤电机），立刻断电 | 立刻 ↓ 减小 Kd |

### Ki（积分增益）

| 状态 | 现象 | 应对 |
|------|------|------|
| **Ki=0** | 无积分，稳态误差无法消除 | 直立环通常 Ki=0 即可 |
| **Ki 过小** | 稳态时轮子有明显持续低速运动 | 调大 Ki |
| **Ki 合适** | 松手后车能缓慢回到中立点 | 当前 Ki=0 ✅ |

## 五、速度环参数推导

### 江协科技速度环经验公式

```
江协科技：速度环 Ki ≈ Kp / 200

本车：SPEED_KP = 0.5 → SPEED_KI ≈ 0.5 / 200 = 0.0025
本车：SPEED_KP = 1.0 → SPEED_KI ≈ 1.0 / 200 = 0.005
```

### 速度环推荐范围

| 参数 | 当前值 | 推荐值 | 说明 |
|------|--------|--------|------|
| **SPEED_KP** | **0.1 ⚠️** | **0.5 ~ 1.0** | 偏小，遥控响应迟钝 |
| **SPEED_KI** | **0.005** | **0.005 ~ 0.015** | 符合公式 ✅ |
| **SPEED_KD** | 0.0 | 0.0 | 不需要 |

**速度环 Kp 判断标准：**
- SPEED_KP 太小：遥控前进指令发了，车速跟不上，响应迟钝
- SPEED_KP 合适：遥控前进 → 车速快速跟踪指令
- SPEED_KP 太大：遥控一停，车立刻猛刹（有顿挫感）

## 六、Ziegler-Nichols 临界比例度法实车操作步骤

> 适配本车量程的操作流程

```
第一步：准备
  → 拆除所有软体负载，固定重心
  → Ki=0, Kd=0, Kp=0（最小值）

第二步：逐步增大 Kp，找临界振荡点
  → 从 Kp=1.0 开始，每次 +0.5
  → 观察：车开始出现等幅低频来回摆动（不倒但不停）
  → 记录这个 Kp 值 = Kp_critical

第三步：按 Z-N 表格求参数
  直立环用 PID 档：
    Kp = Kp_critical × 0.6
    Ki = Kp_critical × 0.4 / 100（换算到本车量程）
    Kd = Kp_critical × 0.075

第四步：上车验证，微调
  → 如果还有振荡 → Kd 微调 +0.02
  → 如果响应迟钝 → Kp 微调 +0.5
```
