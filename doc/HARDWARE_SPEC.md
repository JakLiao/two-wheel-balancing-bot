# 两轮平衡车主控方案 · 硬件规格书

> 基于野火小智 STM32F103C8T6 扩展板
> 目标：学习 PID 调试（直立环 + 速度环）

---

## 一、核心硬件清单

| 模块 | 型号 | 数量 | 备注 |
|------|------|------|------|
| 主控板 | 野火小智扩展板 + STM32F103C8T6 | 1 | |
| 电机驱动 | TB6612FNG | 1 | 双路 H 桥 |
| 电机 | SK11S-48:1 编码器电机 | 2 | 带 AB 相正交编码器 |
| 姿态传感器 | MPU6050 | 1 | I2C 接口 |
| 蓝牙模块 | HC-05 | 1 | 串口通信，手机遥控 |

## 二、引脚分配表（野火小智扩展板 ↔ 各模块）

### 2.1 TB6612FNG 电机驱动

| TB6612 引脚 | STM32 引脚 | 功能 |
|-------------|-----------|------|
| PWMA | PA8（TIM1_CH1） | 左轮 PWM 调速 |
| AIN1 | PB5 | 左轮方向控制 1 |
| AIN2 | PB4 | 左轮方向控制 2 |
| PWMB | PA9（TIM1_CH1） | 右轮 PWM 调速 |
| BIN1 | PB8 | 右轮方向控制 1 |
| BIN2 | PB9 | 右轮方向控制 2 |
| STBY | PB0 | 驱动总使能（必须接） |
| GND | GND | 共地 |
| VM | 7.4V 锂电池正极 | 电机电源 |

### 2.2 编码器电机（SK11S-48:1）

| 电机引脚 | STM32 引脚 | 功能 |
|---------|-----------|------|
| 左轮 A 相 | PA0（TIM2_CH1） | 左轮编码器 A |
| 左轮 B 相 | PA1（TIM2_CH2） | 左轮编码器 B |
| 右轮 A 相 | PA2（TIM2_CH3） | 右轮编码器 A |
| 右轮 B 相 | PA3（TIM2_CH4） | 右轮编码器 B |
| 电机 5V | 3.3V | 编码器供电 |
| 电机 GND | GND | 共地 |

> ⚠️ 编码器必须用 3.3V 供电，严禁接 5V

### 2.3 MPU6050 姿态传感器（I2C）

| MPU6050 | STM32 引脚 | 功能 |
|---------|-----------|------|
| SDA | PB7 | I2C 数据 |
| SCL | PB6 | I2C 时钟 |
| VCC | 3.3V | 供电（严谨 5V） |
| GND | GND | 共地 |

### 2.4 HC-05 蓝牙模块（USART3）

| HC-05 | STM32 引脚 | 功能 |
|-------|-----------|------|
| TXD | PB11（USART3_RX） | 蓝牙数据进单片机 |
| RXD | PB10（USART3_TX） | 单片机发数据给蓝牙 |
| VCC | 5V | HC-05 必须 5V 供电 |
| GND | GND | 共地 |

## 三、共地规则（重中之重）

> 所有模块 GND 全部连在一起，漏接必翻车。

```
STM32 GND
  = TB6612 GND
  = MPU6050 GND
  = HC-05 GND
  = 电池负极
```

| 用电设备 | 电源 | 路径 |
|---------|------|------|

## 五、PID 控制框架

### 两环串联 PID

```
角度环（外环）→ 速度环（内环）→ 电机输出
```

| 环 | 输入 | 输出 | 目标 |
|----|------|------|------|
| **直立环** | MPU6050 俯仰角（°） | 期望速度（°/s） | 车体保持垂直 |
| **速度环** | 编码器速度（脉冲/s） | PWM 占空比 | 速度跟踪 + 增强稳定性 |

> 直立环优先调试，车体能立住后再开速度环
