# PID 参数整定实战指南

> 两轮平衡车 PID 调参核心思路
> 工具：串口打印 + 手机蓝牙遥控

---

## 一、控制架构回顾

```
手机遥控速度期望
     ↓
[速度环 PID]──→ 期望倾角
     ↓
[直立环 PID]──→ PWM 电机输出
     ↓
车体姿态（MPU6050 反馈）
```

---

## 二、调参顺序：先直立环，再速度环

### 第零步：确认传感器正常

在调参前，先用串口监视器确认：

```c
// 在主循环每 100ms 打印一次
printf("Pitch:%+.2f GYRO:%+.2f\r\n", pitch, gyro);
```

**期望现象：**
- 静止时 `Pitch ≈ 0°±2°`（倾斜 ±2° 内正常）
- 用手轻轻推车，`Pitch` 应该快速跟随变化
- `GYRO` 静止时接近 0°/s

如果 `Pitch` 一直飘移，先检查 MPU6050 共地和电源。

## 三、直立环调参（核心！）

**目标：** 车体能自己立住，推一下能自己回正，不倒。

### 步骤 1：调 P（比例）

| 参数 | 初始值 | 操作 |
|------|--------|------|
| Kp | 0 | 从 0 开始 |
| Ki | 0 | 保持 0 |
| Kd | 0 | 保持 0 |

**逐步加大 Kp：**

```
Kp = 10  →  车身有反应但立不住，推一下就倒
Kp = 30  →  车体能立住几秒，推一下能回正一点
Kp = 50  →  车体立得较稳，推一下能弹回来
Kp = 80  →  车身开始抖（振荡）
Kp = 60  →  回到比较稳的状态
```

**找到车体振荡的临界点，记下这个 Kp 值，然后取 60~70% 作为正式 Kp。**

### 步骤 2：加 D（微分，抑制振荡）

| 参数 | 初始值 |
|------|--------|
| Kp | 50（从步骤1选定的值）|
| Ki | 0 |
| Kd | 0 |

**逐步加大 Kd：**

```
Kd = 5   →  振荡略微减小
Kd = 15  →  推一下，车体能快速回正，振荡消失
Kd = 30  →  车身变得"僵硬"，响应变慢
Kd = 20  →  良好平衡点
```

**判断标准：**
- 推一下，车体能快速回正，无明显来回振荡
- 车体在平衡点附近小幅振荡但能稳定住

### 步骤 3：Ki（积分，一般不加）

> 平衡车通常不需要积分项。
> 但如果车体有持续的静态偏斜（总是往一边倒），可以少量加 Ki。

```
Ki = 0.01 → 偏斜减小
Ki = 0.05 → 偏斜消失
Ki = 0.1  → 出现低速抖动
```

**一般 Ki = 0.0 是最常见的选择。**

### 直立环最终参考参数

```
BALANCE_KP = 50~80   （视车重、轮子大小而定）
BALANCE_KI = 0        （一般不加）
BALANCE_KD = 15~30    （需要反复试）
```

## 四、速度环调参

**前置条件：** 直立环调好，车体能稳定立住。

**目标：** 遥控前进/后退时，车速能快速跟踪指令，手松开能停稳。

### 步骤 1：加 P

```
Speed_KP = 0    →  手推车，车速不变（只有直立环在工作）
Speed_KP = 0.5  →  手推车，车速有轻微回正趋势
Speed_KP = 1.0  →  遥控前进/后退，车速跟踪较好
Speed_KP = 2.0  →  速度响应快但松手后有明显回弹振荡
Speed_KP = 1.5  →  较好平衡
```

### 步骤 2：加 I（消除稳态误差）

```
Speed_KI = 0    →  遥控前进后松手，车速缓慢减速，有静差
Speed_KI = 0.01 →  稳态误差减小
Speed_KI = 0.05 →  松手后速度快速归零，效果良好
Speed_KI = 0.1  →  出现振荡
```

## 五、调参问题排查表

| 现象 | 原因 | 解决方案 |
|------|------|---------|
| 车体一直往一边倒 | 传感器没有水平安装 / 重心不在中线 | 机械调平或软件补偿 |
| 直立环 P 很小就抖 | MPU6050 共地不良 / 电机驱动干扰 | 检查 GND 连线，加滤波电容 |
| 速度环加不上，一加就抖 | 直立环不够稳 | 先把直立环调好再加速度环 |
| 编码器数值乱跳 | 电机电源干扰 / 编码器线太长 | 编码器信号线加 100Ω 电阻 |
| 车倒了电机还在转 | 倾角保护阈值太大 | 缩小 BALANCE_ANGLE_MAX |
| 手机指令和实际方向相反 | 电机接线或方向定义反了 | 调换 AIN1/AIN2 或修改代码 |

## 六、进阶调试技巧

### 1. 串口实时绘图
把数据打印成一行，用 Windows Terminal 或 SerialPlot 实时画图：
```c
printf("%+.2f,%+.2f,%d,%d\r\n", pitch, gyro, left_speed, right_speed);
```

### 2. 蓝牙调参（不用每次重烧录）
让 HC-05 接收特殊指令动态修改 PID 参数：
```
SET_KP=60\r\n
SET_KD=25\r\n
GET_STATUS\r\n
```

### 3. 示波器查 PWM
用示波器直接测 PA8/PA9，看 PWM 波形是否正常输出。

## 七、安全注意事项

1. **上车前先绑绳子**，防止车跑出去摔坏
2. **手放在车旁边**，随时准备扶住
3. **先低电压（3.7V）测试**，确认正常再上 7.4V
4. **倾角超过 45° 自动停电机**，防止电机烧毁
5. **不要长时间空转电机**，TB6612FNG 会发热
